Usage

  1. Fork https://gitlab.com/zwartell/zjw-generic-instructions to project repository.
    • ZJW:  https://gitlab.com/gitlab-org/gitlab-foss/-/issues/25805
    • Anyone else: https://docs.gitlab.com/ee/user/project/repository/forking_workflow.html#creating-a-fork
  2. In clone the forked repo and setup a new branch as follows::
    $ git clone Your-Forked-Repo
    $ git remote add upstream https://gitlab.com/zwartell/zjw-generic-instructions
    $ git branch Programming-Language-Tutorial-XYZ
    $ git checkout Programming-Language-Tutorial-XY
    $ git push --set-upstream origin rogramming-Language-Tutorial-XYZ

  3. Develop project specfic features in the branch
  4. If you make updates, bug fixes, improvements relevant to generic the template files, merge them back to fork master branch
  5. Request pull of your changes to master.
